Doce Tentação

O projeto "Doce tentação" tem como objetivo criar uma plataforma online para a apresentação, divulgação e venda de bolos personalizados.
O site visa oferecer uma experiência agradável e intuitiva aos usuários, permitindo que eles escolham bolos para diversas ocasiões, como aniversários, casamentos, festas corporativas, entre outros.
Além disso, o site permitirá que os clientes façam pedidos, personalizem seus bolos com sabores e decoração, e realizem pagamentos de maneira simples e segura.


Link do site:https://tinyurl.com/Doce-Tentacao

Minimalix

O Minimalix é um site projetado para demonstrar o trabalho de um desenvolvedor Front-End, focando na criação de interfaces de usuário com design minimalista e alto desempenho.
O projeto visa exibir um portfólio profissional, destacando habilidades em HTML, CSS, JavaScript e outras ferramentas relevantes, com uma interface limpa, funcional e responsiva.
O objetivo é apresentar o desenvolvimento front-end com ênfase na simplicidade, performance e usabilidade.


Link do site:https://tinyurl.com/Minimalix
